# Setup Auth

[< README](../README.md)

Go to [Firebase console](https://console.firebase.google.com/)
- Select "Authentication"
- Hit "Get Started"
- Select Apple -> Enable -> Save
- Select Google -> Enable 
  - Update public-facing name
  - Add Project support email
  - Hit Save
